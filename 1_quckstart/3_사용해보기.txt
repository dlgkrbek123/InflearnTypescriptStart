------------------------------------------------------------

- nodejs를 설치
- npm init -y로 package.json 생성
- npm install typescript


- 타입스크립트 설정 파일 생성 (tsconfig.json)
- npx tsc --init

------------------------------------------------------------

ts파일 작성 후 아래의 명령어를 입력해 컴파일하면 js파일이 생성

-- npx tsc


- target에 따라 변수선언, 화살표함수등의 문법도 변환된다.
- 타입정보는 컴파일 시에만 활용된다.


- lodash의 경우 라이브러리에서 타입정의 파일을 제공 x
  아래의 명령어로 definitelyTyped의 타입정의 파일을 설치하여 사용!!

-- npm install @types/lodash

------------------------------------------------------------

- typescript는 export가 있으면 해당파일을 모듈로 인식
  모듈의 변수는 해당 파일로 스코프가 제한되어 변수 이름 충돌 x

------------------------------------------------------------

- npx명령어는 node_modules/bin에 있는 바이너리를 실행
